# Contributing and Product Feedback

Thank you for your interest in Mawared ERP.

This repository is a public product showcase. The commercial source code is maintained privately, so this repository does not accept source-code contributions or pull requests that attempt to recreate private product functionality.

## Useful contributions

You are welcome to:

- Report a broken link, typo, or accessibility issue in the showcase.
- Suggest a product capability or workflow.
- Share feedback about Arabic RTL user experience.
- Recommend an HR, attendance, leave, payroll, procurement, or reporting use case.
- Ask a high-level architecture or product question that does not require disclosure of private implementation details.

## Before opening an issue

- Do not include passwords, tokens, credentials, private URLs, employee data, customer data, or confidential company information.
- Do not publish vulnerability details in a public issue. Follow `SECURITY.md` instead.
- Search existing issues to avoid duplicates.
- Explain the business problem and expected outcome, not only the proposed screen or button.

## Pull requests

Small documentation, accessibility, or showcase-site fixes may be considered. Large changes, commercial source-code additions, reverse-engineered features, or changes that conflict with the proprietary license will not be accepted.

## Language

Feedback is welcome in Arabic or English.
